# Dashboard Samples

Sample dashboard YAML files are in `docs/dashboards/`.

Suggested files:
- `transit_test.yaml` - test-driven validation dashboard
- `zagreb_transit_demo.yaml` - demo (HR)
- `zagreb_transit_demo_en.yaml` - demo (EN)
- `zagreb_transit_full.yaml` - full dashboard (HR)
- `zagreb_transit_full_en.yaml` - full dashboard (EN)

Import them in Home Assistant via Dashboard Raw Configuration or copy cards into an existing dashboard.
